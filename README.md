# ProyectoProg4
En este proyecto hay 3 ejecutables: WORDLEJALADAdmin.exe, WORDLEJALADCliente.exe y WORDLEJALADServer.exe.
El primero en ejecutar debe ser el Admin, en el cual aparte de funciones de bases de datos como añadir, editar
o borrar usuarios se debe iniciar la función de cargar CSV con las palabras. El fichero CSV se encuentra en el
propio documento (palabras_ejemplo.csv). 
Los otros dos ejecutables son el cliente y el servidor. Antes de ejecutar el cliente se debe iniciar el servidor
en una consola aparte (ambas deben estar funcionando a la vez). Desde el cliente tendremos la opcion de jugar, que
comunicará la partida al servidor y esté escuchará y responderá en consecuencia. Desde el cliente también se podrán
visualizar y editar los datos de la base de datos que respectan al propio usuario, sin embargo este solo enviará la
orden al servidor, y este se encargará de ejecutar las operaciones.
